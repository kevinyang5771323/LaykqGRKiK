# 基于Django的Python电商产品数据可视化分析平台简介

## 核心功能
- 登陆注册
- 产品信息数据管理
- 产品评论数据管理
- 情感分析可视化
- 用户管理
- 认证授权管理

## 适用场景
- 电商数据分析
- 用户行为研究
- 产品市场调研
- 商业决策支持

## 技术特点
- 采用Django框架，确保系统稳定性和扩展性
- Pyecharts库支持丰富数据可视化效果
- SQLite数据库适用于中小规模数据存储和查询
